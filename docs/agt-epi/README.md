# AGT + .EPI Docs

This section packages the maintainer-facing AGT -> `.epi` proposal as one focused kit.

## What is here

- [Artifact RFC](artifact-rfc.md): short proposal for adding `.epi` as an optional Annex IV export layer
- [Thread reply](thread-reply.md): ready-to-post maintainer reply with the exact demo commands
- [Runnable demo](../../examples/agt-epi-demo/README.md): 2-minute proof using the existing `epi import agt` path
- [Visual proof](../assets/agt-epi-demo-case-view.png): AGT evidence reopened in the EPI viewer

## Positioning

- AGT stays unchanged
- `.epi` is an optional portable artifact layer, not a replacement
- the current prototype is bundle-first
- bare Annex IV JSON ingestion is not claimed in this doc set
