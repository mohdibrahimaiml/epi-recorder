# EPI-Recorder: 5-Minute Enterprise Quickstart

## Workflow (5 min)

```bash
epi annex init
epi annex sign all --key annex
epi annex pack
epi verify annex-iv-compliance.epi --verbose
```

Shows P verification.
## Hand to Auditor

The .epi file is the signed audit trail.
