EPI Documentation

AI evidence infrastructure for consequential AI.

Tamper-evident decision case files for AI governance, model risk, compliance engineering, and AI platform teams.

Install

pip install epi-recorder

Record

python my_workflow.py

Import AGT

epi import agt examples/agt/sample_bundle.json --out sample.epi
epi verify sample.epi
epi view sample.epi

Review

epi view evidence.epi
epi review evidence.epi
epi verify evidence.epi
Decision Case Files One `.epi` record for what happened, which workflow acted, and what evidence still holds.
Policy and Control Evidence Embed `policy.json` and analysis so reviewers can see which controls applied and where a run failed.
Human Review and Signoff Append `review.json` without rewriting the original sealed evidence.
Tamper-Evident Trust Show artifacts as `Signed`, `Unsigned`, or `Tampered` with offline verification.
Transformation Audit Imported AGT artifacts include `artifacts/agt/mapping_report.json` so the evidence conversion itself is inspectable.
