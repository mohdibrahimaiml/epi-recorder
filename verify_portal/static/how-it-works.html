SYSTEM DIAGRAM

HOW EPI
WORKS

From raw execution to a sealed, verifiable evidence package — in five cryptographically verifiable steps.

00 // INSTALL

pip install epi-recorder. Zero system dependencies. macOS, Linux, Windows.

$ pip install epi-recorder

01 // RECORD

Wrap client, open context. LLM calls, tools, decisions captured as structured JSONL.

$ epi run my_agent.py

02 // VERIFY

SHA-256 integrity, Ed25519 signature, prev_hash chain, step completeness, count attestation.

$ epi verify agent.epi

03 // REVIEW

Inspect timeline, check 9-pass fault analyzer, add review notes. All local, zero uploads.

$ epi view agent.epi

04 // SHARE

Send the .epi to auditors. Anyone verifies independently — browser-based, offline.

Drop at epilabs.org/verify
